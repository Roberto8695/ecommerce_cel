Coloca tu archivo de video de smartphones aquí con el nombre "smartphone-video.mp4".

Recomendaciones para el video:
1. Usa formato MP4, idealmente con codec H.264
2. Duración recomendada: 10-20 segundos (puede ser en loop)
3. Resolución ideal: 1080p o 720p
4. Tamaño de archivo: mantenerlo por debajo de 5MB para no afectar la velocidad de carga
5. El contenido debería mostrar smartphones modernos, tecnología móvil o experiencias de usuario con dispositivos móviles

Si no tienes un video disponible, puedes encontrar videos libres de derechos en:
- Pexels (https://www.pexels.com/es-es/videos/)
- Pixabay (https://pixabay.com/es/videos/)
- Coverr (https://coverr.co/)

Busca términos como "smartphone", "mobile phone", "technology" o "digital device".
